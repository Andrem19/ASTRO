# Compatibility Scoring

Compatibility scores are interpretive signals, not absolute truth or a deterministic
forecast.

## Calculation

1. Significant aspects are found between the two natal charts.
2. Each aspect receives a weight based on aspect type and orb tightness.
3. Weighted aspects are grouped by theme.
4. Theme scores are normalized to a 0-100 scale.
5. Overall score combines emotional, mental, romantic, stability, and challenge signals.

## Aspects

Major aspects:

- conjunction
- opposition
- trine
- square
- sextile

Minor aspects are included only when `include_minor_aspects=true`:

- semi-sextile
- semi-square
- quintile
- sesquiquadrate
- quincunx

## Reading Scores

Each score contains `value`, `explanation`, and `supporting_aspects`.

`challenge` is not a bad result. It highlights tension, activation, and possible growth
points.

Astrology is symbolic and interpretive. Scores should support reflective discussion, not
replace personal judgment or professional advice.
